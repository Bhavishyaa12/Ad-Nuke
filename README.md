## Pre-requisite:
- Rooted Android with [Magisk](https://github.com/topjohnwu/Magisk "Official Magisk Repository")!!!
- A working brain
- Termux app installed
- Internet connection
- Magisk systemless hosts module required

‚ö†Ô∏è **Note:**
Please install magisk systemless hosts module in magisk settings to make this script work.
Go to Magisk Settings --> Click on Systemless hosts (REBOOT).
   
## Features:
- Systemlessly install ad blocking hosts files in `/etc/hosts` with one click 
- Easy setup run once and forget
- Optionally blocks all the porn sites by specifying `-p` flag  

‚ö†Ô∏è **Note:**
An internet connection is required to download hosts file so please make sure you have a stable connection before running this script.

## Installation:
- In Termux:
```sh
curl -O https://raw.githubusercontent.com/Bhavishyaa12/Ad-nuke/main/adnuke.sh
```
- Then:
```sh
sh adnuke.sh -h
```
## Enjoy ad free browsing!!!

## Usage:
- To see help file:
```sh
sh adnuke.sh -h
```
- To block ads only:
```sh
sh adnuke.sh -b
```
- To block porn sites:
```sh
sh adnuke.sh -p
```
- To restore default hosts file: 
```sh
sh adnuke.sh -r
```
