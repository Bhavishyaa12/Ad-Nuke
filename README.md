## Usage:
- To see help file:
```sh
sh adnuke.sh -h
```
- To block ads only:
```sh
sh adnuke.sh -b
```
- To block porn sites:
```sh
sh adnuke.sh -p
```
- To restore default hosts file: 
```sh
sh adnuke.sh -r
```
